LLM Endpoint Explorer

Queries an OpenAI-compatible /models endpoint and surfaces every stat it returns — context window, output cap, pricing, modality, ownership. Run a per-model chat round-trip to confirm the full path works.
Runs entirely in the browser. Not every provider populates every field via /models — OpenAI returns only id/created/owned_by; LM Studio returns just id; OpenRouter returns the richest data (context, pricing, modality). Use Test on any model to confirm /chat/completions actually works end-to-end. If a provider blocks cross-origin (Anthropic direct API, Gemini native) you'll see a network error — use a local relay or CLI in that case.
